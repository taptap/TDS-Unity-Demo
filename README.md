## 介绍
当前项目是 TDS Unity 游戏演示 Demo，支持 Android 和 iOS 两个平台。

## 已集成的 TDS 功能
* TapTap 登录、内建账户登录
* 防沉迷
* 成就
* 排行榜
* 内嵌动态
* 公告 -- 仅支持在 iOS 或者 Android 端查看
